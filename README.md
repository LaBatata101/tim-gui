# T.I.M - Triangle Inventory Management
This is the repository for the user interface of the [tim](https://github.com/LaBatata101/tim) project.

## Dependencies
- requests ^2.27.1
- PySide6  ^6.3.0
- pydantic ^1.9.0

### Installing the dependencies
```bash
$ cd tim-gui
$ pip install -r requirements.txt
```
### Running
```bash
$ python main.py
```
